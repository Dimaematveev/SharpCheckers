Sharp Checkers
Created by Tony Okorodudu
Written 2007
Language: C# (.NET Framework 2.0)


SUMMARY AND HISTORY
Sharp Checkers is a checkers game written using the Model View Presenter (MVP) pattern.  I
wrote this some time ago and recently ran into it on my computer.  I initially created it
over the course of one week to experiment with the MVP design pattern.  Up until today, it
was a throwaway project but now I realize it can be expanded upon and used as a learning
tool for myself and others.

Sharp Checkers is nowhere near as feature rich as my old hit, Maverick Checkers, which was
written in Java.  At the time I published Maverick Checkers, it was the strongest and most
feature rich Java application available on the web based upon my web searches and some
reviews articles I found on the web.  Maverick Checkers can be run as a standalone
application or through a browser.  I've included the Maverick Checkers application and
screenshots in the source control repository as a point of reference
(~/resources/MaverickCheckers).

Over time, I hope others will extend this project with more features so that it surpasses
Maverick Checkers.

РЕЗЮМЕ И ИСТОРИЯ
Sharp Checkers - игра в шашки, написанная с использованием шаблона Model View Presenter (MVP). я
написал это некоторое время назад и недавно столкнулся с этим на моем компьютере. Я изначально его создал
в течение одной недели, чтобы экспериментировать с шаблоном проектирования MVP. До сегодняшнего дня это
был одноразовым проектом, но теперь я понимаю, что его можно расширить и использовать в качестве обучения
инструмент для себя и других.

Sharp Checkers далеко не так богаты, как мой старый хит Maverick Checkers, который был
написано на Java. В то время, когда я публиковал Maverick Checkers, он был самым сильным и самым
многофункциональное Java-приложение, доступное в Интернете на основе моих веб-поисков и некоторых
Обзоры статей, которые я нашел в Интернете. Maverick Checkers можно запускать как самостоятельный
приложение или через браузер. Я включил приложение Maverick Checkers и
скриншоты в репозитории исходного кода в качестве ориентира
(~ / ресурсы / MaverickCheckers).

Я надеюсь, что со временем другие расширят этот проект, добавив больше возможностей
Маверик Шекерс.
